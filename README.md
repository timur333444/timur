# Expense Tracker

**Автор:** Абитов Тимур

## Описание программы

Expense Tracker — это графическое приложение для отслеживания личных расходов. Позволяет добавлять, удалять, фильтровать расходы по категории и дате, а также подсчитывать сумму расходов за выбранный период. Данные сохраняются в JSON-файл.

## Требования

- Python 3.7+
- tkinter (входит в стандартную библиотеку Python)

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/expense-tracker.git
cd expense-tracker

# Запуск приложения
python expense_tracker.py

# Запуск тестов
python -m unittest test_expense_tracker.py
Как использовать
Добавление расхода: заполните поля "Сумма", "Категория", "Дата" и нажмите "Добавить".

Удаление: выберите запись в таблице и нажмите "Удалить выбранное".

Фильтрация: выберите категорию или введите дату, нажмите "Применить фильтр".

Подсчёт суммы за период: введите начальную и конечную дату, нажмите "Рассчитать сумму".

Сохранение: данные автоматически сохраняются при добавлении/удалении.

Примеры использования
Пример 1: Добавление расходов
Сумма: 350.50, Категория: Еда, Дата: 2024-12-01 → Добавить

Сумма: 1200, Категория: Транспорт, Дата: 2024-12-02 → Добавить

Пример 2: Фильтрация по категории
Выбрать категорию "Еда" → Применить фильтр → видны только расходы на еду

Пример 3: Подсчёт суммы за декабрь 2024
Начало: 2024-12-01, Конец: 2024-12-31 → Рассчитать сумму

Пример 4: Валидация (ошибочный ввод)
Сумма: -100 → ошибка "Сумма должна быть положительным числом"

Дата: 31.12.2024 → ошибка "Дата должна быть в формате ГГГГ-ММ-ДД"
