# Expense Tracker

**Автор:** Абитов Тимур

## Описание программы

Expense Tracker — это графическое приложение для отслеживания личных расходов. Позволяет добавлять, удалять, фильтровать расходы по категории и дате, а также подсчитывать сумму расходов за выбранный период. Данные сохраняются в JSON-файл.

## Требования

- Python 3.7+
- tkinter (входит в стандартную библиотеку Python)

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/expense-tracker.git
cd expense-tracker

# Запуск приложения
python expense_tracker.py

# Запуск тестов
python -m unittest test_expense_tracker.py
